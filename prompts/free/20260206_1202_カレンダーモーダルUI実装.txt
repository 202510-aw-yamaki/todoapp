【指示】
- index.htmlに日セルクリックでモーダル拡大表示と当日TODO一覧を表示するカレンダーUIを実装
- animejs v4 createLayout を使用し <dialog> をモーダルとして使う
- ?lang= パラメータや既存構成は維持
- /api/todos?date=YYYY-MM-DD で日付指定の取得
- CDN禁止のため animejs UMD を static/lib に配置（手順をREADMEに記載）

【前提】
- Java：17
- Spring Boot：3.x
- Maven：使用
- パッケージ名：com.example.todo
- 画面(MVC)/API：MVC + REST API
- DB：H2

【受け入れ条件（完了条件）】
- カレンダーがH1直下に表示される
- 日セルクリックでモーダルが拡大表示され、当日TODO一覧が表示される
- ESCまたは背景クリックで閉じ、フォーカスが元セルに戻る
- animejs UMD を local に配置して動作する

【非対象（今回はやらないこと）】
- カレンダーの高度な月移動UI
- 本番用の完全なデザイン調整
