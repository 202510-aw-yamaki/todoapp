# 目的
カレンダーモーダル内の文言「【ここをクリックすれば閉じます】」が右上に配置されている。これを左側、または画面中央寄りに配置したい。

# 問題
- クリックして閉じる案内文が右上に固定されていて視線誘導が弱い。
- 右上配置のため、モーダル本文と距離があり不自然。

# 解決方針
- 対象要素のHTML構造を確認し、案内文の要素にクラスを付与する。
- CSSで右上配置の指定（position/float/absolute/top/right など）を解除。
- 左寄せになるように 	ext-align と margin を調整する。
- 必要に応じて display: block と width: 100% を指定して中央揃えを実現。

# 実装指示（Codexに渡すプロンプト例）
以下を実行してください：
1. src/main/resources/templates/calendar.html で「【ここをクリックすれば閉じます】」の要素を特定し、専用クラス（例: calendar-close-hint）を付与してください。
2. src/main/resources/static/css/calendar.css にスタイルを追加してください。
   - 右上固定の指定がある場合は解除。
   - 左寄せにしたい場合: 	ext-align: left; と margin-left を調整。
3. レイアウト崩れがないか確認し、モーダル内の他要素と干渉しないようにしてください。

# 期待結果
案内文「【ここをクリックすれば閉じます】」が右上ではなく左側に配置され、視認性と導線が改善される。
